# VS Code Remote - Containers Sample

## Steps

1. Install [Docker Desktop](https://www.docker.com/products/docker-desktop)
1. Install [Dev Containers](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)
1. Open this directory.
1. Execute `Dev Containers: Reopen in Container` in the VS Code Command Palette.
