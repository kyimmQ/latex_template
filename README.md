# VS Code Remote - Containers Sample

## Steps

1. Install [Docker Desktop](https://www.docker.com/products/docker-desktop)
2. Install [Dev Containers](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)
3. Open this directory.
4. Execute `Dev Containers: Reopen in Container` in the VS Code Command Palette.
5. Update user 'settings.json' based on the provided configuration.
